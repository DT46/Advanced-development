Javascript libraries

Anime.js

Anime.js werkt met alles wat met internet te maken heeft. Het werkt met CSS, SVG, DOM-kenmerken en JavaScript-objecten. Github test.





Chart.js

Met Chart.js kan je staaf- en lijndiagrammen aanpassen om een duidelijk visueel onderscheid te maken tussen datasets. Je kan complexe, eenvoudige datasets op datum, tijd, of zelfs volledig aangepaste schalen maken met gemak.





Aos.js

Met AOS kan je je scroll animeren, zo kun je gebruik maken van verschillende animaties: fades, flips en zooms. Hiermee kan je snel en gemakkelijk fade animaties toevoegen in je HTML.





Parallax.js

Met Parallax kan je je elementen positioneren dat als je op verschillende plekken je muis houdt, het element beweegt.





Slick.js

Met Slick kan je verschillende slideshows maken, hierbij kun je gebruik maken van bepaalde opties zoals: autoplay, faden en swipen.





React.js

Met React.js kan je in je website componenten maken, je kan er gebruikersinterfaces mee bouwen en complexe gebruikersinterfaces samenstellen.





JQuery.js

Bij JQuery kennen ze 'Schrijf minder, doe meer'. Het is een hulpmiddel om het schrijven van veel gebruikte JavaScript taken meer samenvattend maken.





Backbone.js

Backbone.js geeft structuur aan webapplicaties en heeft een grote API met veel functies. Als je een website hebt met veel JavaScript toepassingen kan je het door middel van Backbone.js simpel houden.





Polymer.js

Met Polymer kan je webapplicaties bouwen met behulp van webcomponenten. Je kan elementen zelf helemaal aanpassen. De bibliotheek wordt ontwikkeld door Google-ontwikkelaars en bijdragers op GitHub.





Three.js

Three.js is een API die wordt gebruikt om geanimeerde 3D-graphics te maken en weer te geven in een webbrowser. Je kan bijvoorbeeld met je muis erop klikken en ermee bewegen, dit zorgt voor mooie effecten.





Pixi.js

Met Pixi.js kan je 2D-grafische elementen gebruiken. Je kan bijvoorbeeld op de achtergrond bewegende elementen gebruiken van allerlei soorten en maten. Ze focussen op performance dus Pixi.js is erg snel.
